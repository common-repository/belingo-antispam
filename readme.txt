A simple and effective anti-spam plugin for WordPress.

== Description ==

A simple and effective anti-spam plugin for WordPress. The plugin algorithm analyzes user behavior on the site and, if necessary, marks it as spam, after which all bot actions are blocked. The plugin does not have any complex settings and is very easy to install, just install and activate. That's all! Your WordPress is protected.

== Installation ==

1. Upload `belingoantispam` to the `/wp-content/plugins/` directory.
2. Activate `Belingo.AntiSpam` in your wordpress site if it is not activated already.
